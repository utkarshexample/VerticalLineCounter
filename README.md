VerticalLineCounter
===================
A Java console app that counts the number of vertical black lines in a black and white image.
 
Requirements
------------
Java JDK 8 or higher
 
How to Run
----------
1. Compile:
   javac VerticalLineCounter.java
 
2. Run:
   java VerticalLineCounter <absolute_path_to_image>
 
   Windows: java VerticalLineCounter C:\assignment\img_1.jpg
   Mac/Linux: java VerticalLineCounter /Users/yourname/assignment/img_1.jpg
